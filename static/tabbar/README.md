# TabBar Icons

此目录需要放置底部导航栏图标。每个 Tab 需要两个图标：
- 普通状态（未选中）
- 激活状态（选中，颜色 #C2185B）

## 需要的图标文件：

| 图标 | 未选中文件 | 激活文件 | 说明 |
|------|-----------|---------|------|
| 孕期记录 | home.png (24x24px) | home-active.png (24x24px) | 首页/孕期记录图标 |
| 知识中心 | knowledge.png (24x24px) | knowledge-active.png (24x24px) | 书本/知识图标 |
| 报告管家 | reports.png (24x24px) | reports-active.png (24x24px) | 文档/报告图标 |
| 我的 | profile.png (24x24px) | profile-active.png (24x24px) | 用户/个人中心图标 |

## 图标规范：

- **尺寸**: 建议使用 24x24px 或 48x48px（2x）
- **格式**: PNG，透明背景
- **普通状态**: 灰色 (#757575)
- **激活状态**: 品牌主色 (#C2185B)

## 图标资源推荐：

- [iconfont](https://www.iconfont.cn/) - 阿里巴巴图标库
- [iconpark](https://iconpark.oceanengine.com/) - 字节跳动图标库
- [Material Design Icons](https://fonts.google.com/icons) - Google Material Icons

## 临时解决方案：

在开发阶段，可以使用 uni-app 内置的图标或文字代替，或者使用 base64 格式的 SVG 图标。
